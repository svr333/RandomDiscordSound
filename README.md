# RandomDiscordSound
Small application that deletes all the existing Soundboard sounds in a Discord server and replaces it with random sounds from a set folder.  
To use, you need the following:

- Environment variable `Token` with your Discord bot token
- A folder with random .ogg sounds (max length 5.2s and 512 KB in size)
- Set the right GuildId/Folder in the code at the very top of the Program.cs file
